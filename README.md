# SimpleCinemachine
포트폴리오에서 사용한 간단한 시네머신 기능입니다.  

### 사용된 구역  
MVP 카메라처리  
게임처음 시작시 나오는 UI에 나오는 캐릭터이동, 카메라이동에 사용  
각 스테이지마다 있는 진입씬에 사용  
메인메뉴애서 버튼 선택시 해당 위치로 아동하는 기능에 사용

### 기능  
특정오브젝트를 제어하여 특정위치로 이동, 회전하는 기능을 가짐.  
각 장면마다 시작, 끝시간을 지정하여 이동속도를 지정하며 애니메이션커브로 해당 이동속도를 조절함.  
간단한 에디터지원
